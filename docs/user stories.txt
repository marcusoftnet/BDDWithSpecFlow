Kravbeskrivning
===============
Vision: En personlig filmkatalog för hantering av min filmsamling. 
        Den ska också hålla reda på utlåningar. 
	    En enkel klient från kommandoraden räcker tills vidare.

1
För att få en överblick av min samling 
som filmsamlare 
vill jag kunna se hur många filmer jag har 

2
För att få en överblick av min samling
som filmsamlare 
vill jag kunna lista alla filmer i mitt bibliotek samt se totala antal filmer 

Givet att filmsamlingen innehåller "Alien", "E.T." och "Jaws"
När jag anger kommando: ListaFilmer
Så ska resultatet vara:
| Filmnamn |
| Alien |
| E.T. |
| Jaws |
| Du har 3 filmer i samlingen |

3
För att kunna utöka mitt innehav, t.ex. via kommando "add Alien"
som filmsamlare 
vill jag kunna registrera en ny film till samlingen

4
För att samlingen ska vara korrekt, t.ex. via kommando "add Alien" -> "Filmen finns redan"
som filmsamlare 
vill jag att inga dublettfilmer kan registreras 

5
För att kunna hålla reda på vem som har lånat vilka filmer, t.ex. via kommando "loan E.T.".
som filmsamlare 
vill jag kunna registrera ett lån av en dvd 

6
För att veta vilka filmer jag har fått tillbaka, t.ex. via kommando "return E.T."
som filmsamlare 
vill jag kunna markera återlämning av utlånad film 

7
För att kunna hålla reda på vem som har haft en film länge
som filmsamlare 
vill jag kunna registrera ett lån av en film med datum 

8
För att se vilka filmer som för tillfället är utlånade
som filmsamlare 
vill jag kunna lista alla filmlån 

9
För att öka sannolikheten att jag får tillbaka filmerna
som filmsamlare 
vill jag kunna skicka påminnelser till de låntagare som har haft filmer lånade länge 

10
För att kunna stänga av datorn utan att registret försvinner.
som filmsamlare 
vill jag att filmer och lån ska sparas beständigt (automatiskt)
